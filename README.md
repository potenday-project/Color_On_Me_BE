-   퍼스널 컬러 진단 후 종이나, pdf로 제공되는 진단서가 많이 불편했어요..
-   진단서에는 너무 많은 정보들이 있어서 한눈에 보기에도 어렵고 화장품이나 옷을 살때 활용하기가 힘들었어요...
-   그래서 **핸드폰으로, 필요한 정보만, 간단하게 확인**할 수 있는 서비스를 개발했어요!😎
